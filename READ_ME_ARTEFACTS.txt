Artefact for Bachelor Project

AN EMPIRICAL ANALYSIS OF SNOWFLAKE AS A DATA VISUALISATION BACKEND TO EXPLORE INTERNET BACKGROUND RADIATION
The study conducted a experiment on a raw Dataset 3, capture by using Cloud Telescope.
The dataset was enriched using the developed tools and the workflow defined in the manuscript.
The enriched dataset comprised of 53 million event and approximately 8GB in size. 
Following the enrichment pipeline a Python script generated tabular summaries, which are in form of .tex file.
In this folder both enriched dataset and result.tex file are available for transparency, validation and reproducibility.
The output_enriched.CSV.gz file and result.tex, are in compressed form. There is no password or restriction applied.
These file are submitted as a artefacts of this study.
