# Enrichment and Visualisation of Unsolicited Internet Traffic

## Overview
This repository contains my Bachelor's Cyber Security project completed at Noroff University College.

The project investigates the use of Snowflake as a cloud-native analytical platform for processing, enriching, and visualising large-scale Internet Background Radiation (IBR) datasets containing millions of unsolicited network events.

## Project Objectives
- Reproduce an established IBR enrichment pipeline.
- Process large-scale network datasets.
- Load enriched datasets into Snowflake.
- Generate visualisations using SQL queries.
- Evaluate Snowflake as a backend platform for cybersecurity data analytics.
